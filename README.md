# Xlopok Bot - SAMP MoonLoader Script

Автоматический бот для сбора льна и хлопка на серверах Arizona RP в San Andreas Multiplayer.

## Описание

Этот скрипт предназначен для автоматизации процесса сбора льна и хлопка на серверах Arizona RP. Скрипт имеет удобный графический интерфейс с настройками и работает только на официальных серверах Arizona RP.

## Возможности

- ✅ Автоматический сбор льна
- ✅ Автоматический сбор хлопка
- ✅ Графическое меню управления (ImGui)
- ✅ Гибкие настройки:
  - Включение/отключение сбора льна
  - Включение/отключение сбора хлопка
  - Настройка задержки между сборами (500-5000 мс)
  - Настройка радиуса поиска ресурсов (10-100 метров)
- ✅ Сохранение настроек в INI файл
- ✅ Система уведомлений в чат
- ✅ Защита от использования вне серверов Arizona RP
- ✅ Оптимизированная работа без лагов

## Требования

- **SAMP (San Andreas Multiplayer)** - установленная игра
- **MoonLoader** - версия 0.26 или выше
- **Библиотеки MoonLoader:**
  - mimgui (для графического интерфейса)
  - inicfg (для сохранения настроек)
  - lib.samp.events (для обработки событий)

## Установка

1. Скачайте и установите MoonLoader, если еще не установлен
2. Скопируйте файл `Xlopok.lua` в папку `moonloader` в директории вашей игры GTA San Andreas
3. Запустите игру и подключитесь к серверу Arizona RP
4. Скрипт автоматически загрузится при старте игры

## Использование

### Основная команда

```
/Xlopok
```

Эта команда открывает/закрывает меню управления ботом.

### Меню управления

В меню вы найдете следующие опции:

1. **Автоматический сбор** - главный переключатель для активации/деактивации бота
2. **Собирать лён** - включить/выключить сбор льна
3. **Собирать хлопок** - включить/выключить сбор хлопка
4. **Задержка сбора** - время ожидания между сборами (в миллисекундах)
5. **Радиус поиска** - расстояние, на котором бот ищет ресурсы (в метрах)
6. **Сохранить настройки** - сохранить текущие настройки в файл
7. **Закрыть меню** - закрыть окно управления

### Настройка параметров

#### Задержка сбора
- Минимум: 500 мс (быстрый сбор)
- Максимум: 5000 мс (медленный сбор)
- Рекомендуется: 1000-2000 мс для стабильной работы

#### Радиус поиска
- Минимум: 10 метров
- Максимум: 100 метров
- Рекомендуется: 50 метров для оптимального баланса

## Как работает бот

1. Скрипт постоянно сканирует окружающее пространство на наличие объектов льна и хлопка
2. При обнаружении ресурса в заданном радиусе, персонаж автоматически подходит к нему
3. Отправляется команда `/collect` для сбора ресурса
4. После сбора бот ожидает заданное время задержки перед следующим поиском
5. Процесс повторяется до отключения автоматического сбора

## Файлы конфигурации

Настройки сохраняются в файл `Xlopok.ini` в папке `moonloader/config/`:

```ini
[settings]
flax_enabled=1
cotton_enabled=1
collect_delay=1000
search_radius=50.0
auto_collect=0
```

## Устранение проблем

### Скрипт не загружается
- Убедитесь, что MoonLoader установлен корректно
- Проверьте наличие всех необходимых библиотек
- Проверьте лог файл `moonloader.log`

### Бот не собирает ресурсы
- Убедитесь, что включен "Автоматический сбор"
- Проверьте, включен ли сбор нужного типа ресурса (лён/хлопок)
- Увеличьте радиус поиска
- Убедитесь, что вы находитесь рядом с ресурсами

### Меню не открывается
- Проверьте правильность ввода команды: `/Xlopok` (с заглавной буквы X)
- Перезагрузите скрипт командой `/reloadscripts`

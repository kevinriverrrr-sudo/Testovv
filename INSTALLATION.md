# Руководство по установке Xlopok Bot

Подробная инструкция по установке и настройке бота для сбора льна и хлопка на серверах Arizona RP.

## Шаг 1: Предварительные требования

### Установка MoonLoader

1. Скачайте последнюю версию MoonLoader с официального сайта: https://www.blast.hk/threads/13305/
2. Распакуйте архив MoonLoader в папку с игрой GTA San Andreas
3. Запустите игру один раз, чтобы MoonLoader инициализировался

### Проверка установки MoonLoader

После первого запуска в папке игры должна появиться папка `moonloader` со следующей структурой:
```
GTA San Andreas/
├── moonloader/
│   ├── lib/
│   ├── config/
│   └── moonloader.log
```

## Шаг 2: Установка необходимых библиотек

### mimgui (для графического интерфейса)

1. Скачайте библиотеку mimgui: https://www.blast.hk/threads/66959/
2. Распакуйте файлы в папку `moonloader/lib/`

### Другие библиотеки

Следующие библиотеки обычно идут в комплекте с MoonLoader:
- `inicfg` - для работы с INI файлами
- `lib.samp.events` - для обработки событий SAMP

Если их нет, скачайте с официального репозитория MoonLoader.

## Шаг 3: Установка скрипта Xlopok Bot

1. Скопируйте файл `Xlopok.lua` в папку `moonloader` вашей игры
2. Полный путь должен быть: `GTA San Andreas/moonloader/Xlopok.lua`

## Шаг 4: Первый запуск

1. Запустите GTA San Andreas
2. Подключитесь к любому серверу Arizona RP
3. Если все установлено правильно, вы увидите сообщение в чате:
   ```
   [Xlopok Bot] Успешно загружен! Используйте /Xlopok для открытия меню
   ```

## Шаг 5: Настройка бота

1. Введите команду `/Xlopok` в чат
2. Откроется меню управления ботом
3. Настройте параметры по своему усмотрению:
   - Включите/выключите сбор льна
   - Включите/выключите сбор хлопка
   - Установите задержку между сборами (рекомендуется 1000-2000 мс)
   - Установите радиус поиска (рекомендуется 50 метров)
4. Нажмите "Сохранить настройки"

## Шаг 6: Использование бота

1. Отправляйтесь на поле со льном или хлопком
2. Откройте меню командой `/Xlopok`
3. Включите "Автоматический сбор"
4. Бот начнет автоматически собирать ресурсы

## Структура файлов после установки

```
GTA San Andreas/
├── moonloader/
│   ├── Xlopok.lua              # Основной скрипт
│   ├── config/
│   │   └── Xlopok.ini          # Настройки (создается автоматически)
│   ├── lib/
│   │   ├── mimgui.lua
│   │   ├── inicfg.lua
│   │   └── samp/
│   │       └── events.lua
│   └── moonloader.log
```

## Проверка работоспособности

### Проверка загрузки скрипта

1. Откройте файл `moonloader.log` в папке `moonloader`
2. Найдите строки, связанные с Xlopok Bot
3. Убедитесь, что нет ошибок

### Проверка библиотек

Если скрипт не загружается, проверьте наличие всех библиотек:
```
moonloader/lib/mimgui.lua          - должен существовать
moonloader/lib/inicfg.lua          - должен существовать
moonloader/lib/samp/events.lua     - должен существовать
```

## Решение проблем

### Ошибка: "module 'mimgui' not found"

**Решение:**
1. Скачайте библиотеку mimgui
2. Поместите файл `mimgui.lua` в папку `moonloader/lib/`
3. Перезапустите игру

### Ошибка: "module 'lib.samp.events' not found"

**Решение:**
1. Обновите MoonLoader до последней версии
2. Убедитесь, что файл `events.lua` находится в `moonloader/lib/samp/`

### Скрипт загружается, но меню не открывается

**Решение:**
1. Убедитесь, что вы правильно вводите команду: `/Xlopok` (с заглавной X)
2. Попробуйте перезагрузить скрипт: `/reloadscripts`
3. Проверьте, что библиотека mimgui установлена корректно

### Бот не работает на сервере

**Решение:**
1. Убедитесь, что вы играете на сервере Arizona RP
2. Скрипт работает только на официальных серверах Arizona RP
3. Проверьте IP сервера в логе moonloader.log

### Персонаж не подходит к ресурсам

**Решение:**
1. Увеличьте радиус поиска в настройках
2. Убедитесь, что рядом есть ресурсы для сбора
3. Проверьте, что включен нужный тип сбора (лён/хлопок)

## Дополнительные советы

### Оптимальные настройки

Для стабильной работы рекомендуются следующие параметры:
- **Задержка сбора:** 1500 мс
- **Радиус поиска:** 50 метров
- **Включить оба типа:** лён и хлопок

### Безопасное использование

1. Не используйте слишком малую задержку (менее 1000 мс)
2. Периодически проверяйте работу бота
3. Не оставляйте бот работать без присмотра надолго
4. Используйте реалистичные настройки для имитации человеческого поведения

### Горячие клавиши

- `/Xlopok` - открыть/закрыть меню
- `/reloadscripts` - перезагрузить все скрипты MoonLoader

## Обновление скрипта

Для обновления скрипта на новую версию:
1. Сохраните файл настроек `Xlopok.ini` (если хотите сохранить настройки)
2. Удалите старый файл `Xlopok.lua`
3. Скопируйте новый файл `Xlopok.lua` в папку `moonloader`
4. Перезапустите игру или используйте `/reloadscripts`

## Удаление скрипта

Для удаления скрипта:
1. Удалите файл `moonloader/Xlopok.lua`
2. Удалите файл `moonloader/config/Xlopok.ini`
3. Перезапустите игру или используйте `/reloadscripts`

## Поддержка

Если у вас возникли проблемы с установкой или использованием скрипта:
1. Проверьте раздел "Решение проблем" выше
2. Убедитесь, что все библиотеки установлены
3. Проверьте файл `moonloader.log` на наличие ошибок
4. Убедитесь, что используете последнюю версию MoonLoader

## Системные требования

- **Операционная система:** Windows 7/8/10/11
- **GTA San Andreas:** любая версия, совместимая с SAMP
- **SAMP:** версия 0.3.7 или выше
- **MoonLoader:** версия 0.26 или выше
- **RAM:** минимум 2 ГБ
- **Процессор:** любой современный процессор

## Заключение

После выполнения всех шагов скрипт Xlopok Bot должен быть полностью функционален. Если у вас остались вопросы или проблемы, обратитесь к разделу "Устранение проблем" в основном README.md.

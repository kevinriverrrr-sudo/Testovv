# Changelog

Все заметные изменения в проекте Gang Helper будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2024-11-10

### Добавлено
- Первый релиз Gang Helper для Arizona RP
- GUI меню на основе mimgui с тремя вкладками:
  - Вкладка "Команды банды" с кнопками быстрого доступа
  - Вкладка "Отыгровки" для настройки RP текстов
  - Вкладка "Настройки" для общих параметров
- Поддержка 10 банд-команд:
  - `/capture` - начать войну за территорию
  - `/captstats` - статистика войн
  - `/putmoney` - положить деньги на счёт
  - `/getmoney` - снять деньги со счёта
  - `/setdbuy` - установить цену закупки
  - `/setdsell` - установить цену продажи
  - `/stuff` - набить татуировку
  - `/unstuff` - свести татуировку
  - `/dropmats` - выгрузить материалы
  - `/sellgangzone` - продать территорию
- Автоматические отыгровки до и после команд
- Система конфигурации на основе JSON
- До 5 отыгровок до команды и 5 после для каждой команды
- Настраиваемая задержка между отыгровками (500-5000 мс)
- Логирование всех действий хелпера
- Возможность включить/отключить:
  - Весь хелпер
  - Автоматические отыгровки
  - Логирование
- Команда `/gang` для открытия меню
- Конфигурационный файл с отыгровками по умолчанию
- Документация пользователя (README.md)
- Техническая документация (TECHNICAL.md)

### Безопасность
- Локальное хранение всех данных
- Отсутствие внешних запросов
- Открытый формат конфига (JSON)

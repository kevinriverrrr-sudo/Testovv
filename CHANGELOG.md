# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

## [1.0.0] - 2024-11-09

### Добавлено

#### Основные функции
- ✅ Полная интеграция с Funpay.com через content script
- ✅ Анализ конкурентов в реальном времени
- ✅ Система рекомендаций цен с интеллектуальными алгоритмами
- ✅ Аутентификация через cookies Funpay
- ✅ Кроссбраузерная совместимость (Chrome, Firefox, Edge)

#### Анализ конкурентов
- Автоматический парсинг данных конкурентов со страниц Funpay
- Сбор цен, продавцов и информации о товарах
- Расчет статистики: минимум, максимум, среднее, медиана, квартили
- Распределение цен по диапазонам
- Оценка уверенности рекомендаций

#### Рекомендации цен
- Взвешенный алгоритм расчета оптимальной цены
- 4 стратегии ценообразования:
  - Агрессивная (ниже минимума)
  - Конкурентная (ниже средней)
  - Сбалансированная (около медианы)
  - Премиум (выше средней)
- Учет количества конкурентов
- Корректировка на основе вариации цен

#### Интерфейс
- Кнопка "Анализ цен" на страницах товаров Funpay
- Всплывающая панель с результатами анализа
- Popup для проверки авторизации и быстрого доступа
- Страница настроек с полной кастомизацией
- Уведомления о статусе операций
- Адаптивный дизайн

#### Настройки
- Автообновление данных конкурентов
- Настройка интервала обновления
- Выбор количества конкурентов для анализа
- Выбор стратегии по умолчанию
- Корректировка цены (%)
- Минимальный порог цены
- Управление уведомлениями
- Позиционирование кнопки на сайте
- Выбор темы оформления
- Управление историей анализов

#### Безопасность
- Безопасная работа с cookies
- Локальное хранение всех данных
- Минимальные необходимые разрешения
- Отсутствие отправки данных на сторонние серверы
- Проверка авторизации перед доступом к функциям

#### Техническая реализация
- Manifest V3 для современных браузеров
- Service Worker для фоновых задач
- Content Script для интеграции с Funpay
- Chrome Storage API для хранения данных
- Chrome Cookies API для аутентификации
- Модульная архитектура кода

### Технические детали

funpay-analyzer/

#### Используемые API
- Chrome Runtime API
- Chrome Storage API
- Chrome Cookies API
- Chrome Tabs API
- Chrome Scripting API

#### Поддерживаемые браузеры
- Google Chrome 88+
- Microsoft Edge 88+
- Firefox 109+ (с поддержкой Manifest V3)

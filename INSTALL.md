# Инструкция по установке

## Установка расширения Funpay Competitor Analyzer

### Для Chrome/Edge

1. **Скачайте исходный код**
   ```bash
   git clone <repository-url>
   cd funpay-analyzer
   ```

2. **Откройте страницу расширений**
   - В Chrome: перейдите на `chrome://extensions/`
   - В Edge: перейдите на `edge://extensions/`

3. **Включите режим разработчика**
   - Найдите переключатель "Режим разработчика" (Developer mode) в правом верхнем углу
   - Включите его

4. **Загрузите расширение**
   - Нажмите кнопку "Загрузить распакованное расширение" (Load unpacked)
   - Выберите папку с проектом
   - Расширение появится в списке установленных

5. **Проверьте установку**
   - Иконка расширения должна появиться на панели инструментов
   - Нажмите на иконку, чтобы открыть popup

### Для Firefox

1. **Скачайте исходный код**
   ```bash
   git clone <repository-url>
   cd funpay-analyzer
   ```

2. **Откройте страницу отладки**
   - Перейдите на `about:debugging` в Firefox
   - Выберите "This Firefox" в левом меню

3. **Загрузите временное дополнение**
   - Нажмите кнопку "Load Temporary Add-on"
   - Выберите файл `manifest.json` из папки проекта

4. **Проверьте установку**
   - Расширение появится в списке временных дополнений
   - Иконка появится на панели инструментов

### Первая настройка

1. **Авторизация на Funpay**
   - Откройте [Funpay.com](https://funpay.com) в новой вкладке
   - Войдите в свой аккаунт если ещё не вошли

2. **Проверьте авторизацию в расширении**
   - Кликните на иконку расширения
   - Нажмите "Проверить авторизацию"
   - Статус должен измениться на "Авторизован"

3. **Настройте параметры (опционально)**
   - Нажмите "Настройки" в popup
   - Настройте интервал обновления, стратегию цен и другие параметры
   - Сохраните настройки

### Использование

1. **Анализ конкурентов**
   - Откройте любую страницу товара на Funpay.com
   - Найдите кнопку "Анализ цен" в правом верхнем углу
   - Нажмите её для начала анализа

2. **Просмотр рекомендаций**
   - После анализа откроется панель с результатами
   - Изучите рекомендуемую цену и статистику
   - Скопируйте цену или обновите данные

### Устранение проблем

#### Расширение не видит авторизацию

1. Убедитесь, что вы вошли на Funpay.com
2. Очистите куки и войдите заново
3. Нажмите "Проверить авторизацию" в расширении

#### Кнопка не появляется на сайте

1. Обновите страницу (F5)
2. Проверьте, что вы находитесь на странице товара
3. Проверьте консоль браузера на наличие ошибок (F12)

#### Анализ не работает

1. Проверьте статус авторизации
2. Убедитесь, что на странице есть конкуренты
3. Попробуйте другую страницу товара

### Обновление расширения

#### Chrome/Edge
1. Перейдите на `chrome://extensions/` или `edge://extensions/`
2. Найдите расширение в списке
3. Нажмите кнопку "Обновить" (круговая стрелка)

#### Firefox
1. Перейдите на `about:debugging`
2. Нажмите "Reload" напротив расширения

### Удаление расширения

#### Chrome/Edge
1. Перейдите на `chrome://extensions/` или `edge://extensions/`
2. Найдите расширение
3. Нажмите "Удалить"

#### Firefox
1. Перейдите на `about:debugging`
2. Нажмите "Remove" напротив расширения

---

**Примечание**: Расширение работает только на страницах Funpay.com и требует активной сессии для полной функциональности.
